EDK3 Graphics Engine

Graphics Engine that uses a wrapper around OpenGL, made as a project to introduce ourselves into engine programming as a whole.

This project served as a in introduction for lighting techniques, mesh generation, texturing techniques, resource management, and post processing effects.

The Engine also incorporates a small lerping library implementation written from scratch, as a well as cubemapping.
